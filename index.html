Học Tư Duy Thuật Toán & Code

Đây là tài liệu học thuật giúp ông bạn hiểu bản chất của việc lập trình: từ tư duy giải bài toán, cấu trúc đề thi, cách máy chấm hoạt động, cho tới các phương pháp thuật toán cốt lõi bằng C++ và Python.

I. Giới thiệu về Code

C++

C++ là ngôn ngữ lập trình mạnh mẽ, có hiệu năng rất cao, được xây dựng trên nền tảng của ngôn ngữ C và bổ sung thêm nhiều tính năng hiện đại như lập trình hướng đối tượng, lập trình tổng quát (template), và thư viện chuẩn phong phú.

Nhờ khả năng kiểm soát bộ nhớ tốt, tốc độ thực thi nhanh và độ ổn định cao, C++ trở thành lựa chọn hàng đầu trong thi đấu thuật toán, lập trình thi học sinh giỏi, ICPC, Olympiad Tin học, cũng như các hệ thống yêu cầu hiệu suất lớn.

Trong các bài toán thuật toán, C++ cho phép xử lý dữ liệu lớn, vòng lặp phức tạp, và cấu trúc dữ liệu nâng cao (như vector, set, map, priority_queue, …) một cách hiệu quả. Thư viện STL (Standard Template Library) giúp người lập trình triển khai ý tưởng nhanh chóng mà vẫn đảm bảo tối ưu thời gian và bộ nhớ.

Bên cạnh đó, C++ còn được sử dụng rộng rãi trong phát triển game, hệ điều hành, engine đồ họa, phần mềm nhúng và nhiều hệ thống cốt lõi khác, cho thấy tính linh hoạt và sức mạnh lâu dài của ngôn ngữ này.

Tuy cú pháp của C++ tương đối chặt chẽ và khó tiếp cận hơn so với một số ngôn ngữ khác, nhưng khi đã nắm vững, người học sẽ hình thành được tư duy lập trình logic, tư duy tối ưu, và nền tảng vững chắc để tiếp cận các ngôn ngữ cũng như công nghệ nâng cao sau này.

Python

Python là ngôn ngữ lập trình có cú pháp đơn giản, dễ đọc và dễ hiểu, giúp người học nhanh chóng tiếp cận tư duy lập trình mà không bị rào cản bởi cú pháp phức tạp. Nhờ cách viết ngắn gọn và rõ ràng, Python cho phép viết code rất nhanh, đặc biệt phù hợp cho việc prototyping (xây dựng và kiểm thử ý tưởng ban đầu).

Python thường được sử dụng trong các bài toán không yêu cầu hiệu năng quá cao, nơi tính dễ đọc và tốc độ phát triển được ưu tiên hơn tốc độ thực thi. Với hệ sinh thái thư viện phong phú, Python hỗ trợ tốt cho nhiều lĩnh vực như xử lý dữ liệu, trí tuệ nhân tạo, machine learning, tự động hóa, và khoa học máy tính ứng dụng.

Trong học tập và giải bài tập thuật toán, Python giúp người học tập trung vào ý tưởng và phương pháp giải, thay vì phải lo nhiều đến quản lý bộ nhớ hay các chi tiết kỹ thuật thấp. Điều này khiến Python trở thành lựa chọn rất tốt cho người mới bắt đầu, cũng như cho các bài toán có độ phức tạp vừa phải.

Tuy tốc độ chạy của Python chậm hơn so với C++, nhưng với những bài toán phù hợp, Python vẫn mang lại hiệu quả cao nhờ tính linh hoạt, khả năng mở rộng, và thời gian phát triển ngắn.

II. Cấu trúc đề / Script

1. Tư duy về bài

Bài toán lập trình không phải học thuộc lời giải, mà là học cách:

  • Phân tích dữ liệu vào – ra
  • Nhận diện mô hình bài toán
  • Tìm con đường giải tối ưu

2. Phương pháp chấm bài

a) Giới hạn thời gian

Mỗi bài đều có thời gian chạy tối đa (thường 1–2 giây).

b) Máy chấm bài

Máy chấm so sánh output với đáp án chuẩn.

c) Subtask

Mỗi subtask có giới hạn nhỏ hơn để ăn điểm từng phần.

d) Độ phức tạp

Độ phức tạp (complexity) dùng để đánh giá mức độ tốn tài nguyên của một thuật toán khi kích thước dữ liều đầu vào tăng lên.

Thông thường ta quan tâm đến hai loại:

  • Độ phúc tạp thời gian: thuật toán chạy bao nhiêu phép toán.
  • Độ phức tạp bộ nhớ: thuật toán dùng bao nhiêu bộ nhớ

Tại sao cần quan tâm đến độ phức tạp?

Máy tính hay bài toán có giới hạn về thời gian chạy (thời ở khoảng 1-2 giây)

Nếu thuật toán quá chậm, dù code đúng nhưng vẫn bị TLE (Time Limit Exceeded) và máy chấm sẽ không tính điểm của task đó.

Vì vậy, mục tiêu của việc thiết kế thuật toán là giảm độ phức tạp xuống mức có thể chạy được trong thời gian giới hạn.

III. Phương pháp làm bài

Hai con trỏ (Two Pointers)

1. Khái niệm

Dùng hai con trỏ để duyệt mảng nhằm giảm độ phức tạp.

2. Ví dụ

Cho mảng a có n phần tử, kiểm tra cặp số có tổng bằng X trong mảng đã sắp xếp.

Ý tưởng làm bài

Khởi tạo hai con trỏ:
- Con trỏ trái (l = 0) bắt đầu từ đầu mảng
- Con trỏ phải (r = n-1) bắt đầu từ cuối mảng
Trong khi l < r:
- Tính tổng a[l] + a[r]
- Nếu tổng == X → tìm được đáp án, dừng
- Nếu tổng < X → tăng l (cần tổng lớn hơn)
- Nếu tổng > X → giảm r (cần tổng nhỏ hơn)
Kết thúc vòng lặp mà chưa tìm được → không tồn tại cặp phù hợp

Độ phức tạp: O(n) do mỗi con trỏ chỉ đi tối đa n bước
n = 4
a = [1, 3, 5, 10]:
x = 8

l=0 (1), r=3 (10) → 11 > 8 → r--
l=0 (1), r=2 (5)  → 6  < 8 → l++
l=1 (3), r=2 (5)  → 8 ✓

3. Các dạng

  • Ngược chiều
  • Cùng chiều (Sliding Window)
  • Fast – Slow

4. Code

▶ Xem code C++ & Python
#include 
using namespace std;

bool twoSum(vector& a, int X) {
    int l = 0, r = a.size() - 1;
    while (l < r) {
        int s = a[l] + a[r];
        if (s == X) return true;
        if (s < X) l++;
        else r--;
    }
    return false;
}

def two_sum(arr, X):
    l, r = 0, len(arr) - 1
    while l < r:
        s = arr[l] + arr[r]
        if s == X:
            return True
        if s < X:
            l += 1
        else:
            r -= 1
    return False

5. Mẹo & từ khóa

Từ khóa: mảng đã sắp xếp, tìm cặp, đoạn con liên tiếp, tối ưu O(n).
